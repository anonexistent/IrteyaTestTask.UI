# IrteyaTestTask.UI
Тестовое задание для компании ИРТЕЯ. Blazor Web App

> 1. Создать solution с проектом из стандартного шаблона Blazor Web App
> 2. Изменить заполнение таблицы погоды на странице Weather на данные из обособленного "Источника" (на выбор: БД, WebAPI и т.п.). Возвращаемые данные должны быть одинаковы для двух отдельно стоящих запросов.
> 3. Создать solution с тестовым проектом.
>    - в проекте добавить тест, который парсит данные со страницы Weather и сопоставлять с данными из "Источника"
>    - парсер табличных данных умеет работать с любыми моделями данных (построенные на примитивах и других встроенные ValueType)
>    - использовать Page Object Model
>    - проект с тестами поддерживает настройку конфигурации источника данных и вебДрайвера
> 4. Добавить тест проверяющий навигацию по сайту из шаблона
> 
> Общие требования к оформлению:
> - Ведение git'а
> - Два отдельных solution'а
> - Использовать: NUnit, Selenium и AllureReport
> - Готовое задание нужно отдать в zip архиве (исключить папки bin, obj и т.п. из файлов проекта), а также результаты успешного прогона всех тестов Allure
